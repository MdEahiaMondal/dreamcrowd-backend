### Reviews & Ratings ফিচারের পুরো ডিটেইলস (মিটিং অনুযায়ী)

#### Buyer পক্ষ থেকে
1. কখন রিভিউ দিতে পারবে?
    - Class বা Freelance service "Completed" বা "Delivered" স্ট্যাটাসে গেলে।
    - Video course এর ক্ষেত্রে "Mark as satisfactory/unsatisfactory" → unsatisfactory হলে রিভিউ + refund অপশন আসবে।

2. Buyer কী কী করতে পারবে?
    - রিভিউ + রেটিং (1–5 স্টার) দিতে পারবে
    - রিভিউ delete করতে পারবে (যেকোনো সময়)
    - রিভিউ edit করতে পারবে শুধু নিচের কন্ডিশনে:
        - Seller এখনো reply করেনি অথবা
        - রিভিউ পোস্ট করার 24 ঘণ্টার মধ্যে

3. রিভিউ কোথায় দেখা যাবে?
    - Seller এর public profile এ dedicated "Reviews" সেকশনে
    - Buyer এর নিজের ড্যাশবোর্ড → Completed/Delivered ট্যাব থেকে

#### Seller পক্ষ থেকে
1. Seller কী কী করতে পারবে?
    - Buyer এর রিভিউ দেখতে পারবে
    - Reply দিতে পারবে
    - নিজের reply যেকোনো সময় edit বা delete করতে পারবে (কোনো টাইম লিমিট নেই)
    - Buyer এর রিভিউতে abusive language বা মিথ্যা অভিযোগ থাকলে admin কে report করে delete করাতে পারবে

#### Admin পক্ষ থেকে
- সব রিভিউ দেখতে পারবে
- কোনো রিভিউতে abusive language বা policy violation থাকলে সরাসরি delete করতে পারবে
- Seller report করলে admin investigate করে delete করতে পারে

#### অন্যান্য গুরুত্বপূর্ণ পয়েন্ট
- রিভিউ + রেটিং এভারেজ Seller card এ দেখাবে এবং Top Seller criteria তে কাজ করবে (4.5+ রেটিং হলে স্পেশাল আইকন)
- রিভিউ সেকশনের ডিজাইন Figma তে আছে + ফ্রন্টএন্ড সোর্স কোডেও আছে (ক্লায়েন্ট পাঠাবে)

সংক্ষেপে একটা টেবিলে:

| Action                  | Buyer                 | Seller                  | Admin            |
|-------------------------|-----------------------|-------------------------|------------------|
| Give Review + Rating    | Yes                   | No                      | No               |
| Delete own Review       | Yes (anytime)         | No                      | Yes (any review) |
| Edit own Review         | Yes (24h or no reply) | No                      | No               |
| Reply to Review         | No                    | Yes                     | No               |
| Edit/Delete own Reply   | -                     | Yes (anytime)           | No               |
| Report abusive review   | No                    | Yes → admin             | -                |
| Delete any review       | No                    | No                      | Yes              |
